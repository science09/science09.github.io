Android

按钮监听事件的4种实现方法：

  1. 自定义一个ClickListener事件类
  2. 采用匿名的内部类创建点击事件
  3. 让Activity实现点击事件的接口 （常用方法）
  4. 直接在XML描述文件中定义按钮的监听事件

    ADB常用命令

  5. 安装apk到模拟器:
    adb -s emulator -5554 install Path

    说明：在新的AndroidSDK版本中，可以不用指定 -s -PortNumber 这两个参数，直接使用命令: adb install Apk_Path

  6. 卸载一个apk软件:
    adb uninstall <包名>
  7. 进入手机命令行终端:
    adb shell
  8. 查看目前的adb设备：
    adb devices
